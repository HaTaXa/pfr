О проекте «Мониторинг заявок ФГИС "ФРИ" в АСТП»

Принятый руководством принцип работы

Идея руководства МИЦ ОВИВ. Создать один общий файл по ведению хода состояния заявок совместно с карточками на основе динамических ссылок, связанных с ярлыками из того же файла посредством ручной подготовки каждой карточки для каждой заявки.

Далее демонстрируется общий вид и принцип подхода к работе по мониторингу заявок в АСТП, который по мнению руководства МИЦ ОВИВ оценивался и считался наиболее приемлемым.

1. Файл формата Microsoft Office Excel представляет собой один общий процесс хода работы по мониторингу заявок в АСТП, где ярлык «Мониторинг заявок» содержит отображаемую информацию по всем данным о заявках из других ярлыков с присвоенными именами по «номеру» заявки, с учетом ручного ведения, а так же ежедневного обновления ярлыков с заявками каждым ответственным лицом за свой "блок" рабочего процесса по мониторингу заявок в АСТП.

2. В данном файле создаются заявки на основе ярлыка карточки-шаблона по требованию лица, ответственного за ведение жизненного цикла заявки (далее – ЖЦ заявки). Ярлык «Карточка» подготавливается лицом, ответственным в свою очередь за ярлык «Мониторинг заявок» со всей информацией, посредством ручной подготовки гипертекстовой связи (ссылками) из ярлыка «"Карточка" с будущей заявкой под своим номером» в ярлык «Мониторинг заявок». После чего, ответственным лицом за ведение ЖЦ заявки, информация о ней переносится в подготовленную карточку путем заполнения ее данными из АСТП и последующим обновлением информации по заявке.

Однако, это не гарантирует факт ошибочного отображения данных в ярлыке, отвечающим за содержание общих сведений о ходе состояния заявок.

Пояснение:

– Для штатных сотрудников работа по п.2 носит запутанный характер с неоднократными, смешанными между собой процессами начиная от создания карточки на основе шаблона до занесения информации в будущую заявку.

– Для руководства же идеология ведения ЖЦ заявки сводится лишь к отображению шагов прохождения, выраженных в схематично-цветовой гамме ЖЦ в каждой заявке.
