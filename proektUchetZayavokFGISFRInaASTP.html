«Учет заявок ФГИС "ФРИ" на АСТП»

Проект «Учет заявок ФГИС "ФРИ" на АСТП» – это база данных мониторинга статуса и состояния заявок, созданных в ФГИС «ФРИ» – единой базе данных для граждан, признанных в установленном порядке инвалидами, и для органов власти, которые оказывают услуги или меры социальной поддержки инвалидам.

Проект базы данных разработан для мониторинга работы в Автоматизированной системе техподдержки пользователей (АСТП). База данных отражает статистику объема заявок и рабочих заданий для межведомственных подразделений ПФР МИЦ ОВИВ в виде дополнительных заданий в АСТП.

Проект базы данных разработан прежде всего как внутреннее программное обеспечение (ПО)*, позволяющее вести удобный сбор информации и аналитику для всего штатного персонала МИЦ ОВИВ, связанного с работой АСТП.

*Приложение разработано на программном обеспечении Microsoft Office Access 2010, с поддержкой языка программирования Visual Basic for Applications v.7.0.

    Помимо основных принципов ведения статистических данных и работы в целом проект базы данных разработан по правилам, действующим в период на январь-март 2017 года с учетом требований руководства МИЦ ОВИВ:
  • Работа с заявками имеет индивидуальный характер.
    Работа в базе ведется под учетными записями на уровне пользовательского доступа как «Исполнителя» в подчинении у «Куратора». База данных содержит несколько уровней доступа;
  • Рабочее задание на МИЦ ОВИВ, позволяющее учитываться как отдельное рабочее задание;
  • "Светофор" – автоматизированное отображение цветовой гаммой сроков по исполнению как самих заявок, так и рабочих заданий.

База данных имеет интуитивно понятный интерфейс с разработанной пользовательской лентой (Ribbon) – надстройка на панеле инструментов, базирующейся на основе языка XML.

Код адаптирован для функций WinAPI под версии VBA 7.0 и более ранних версий, а так же для 32-х/64-битных систем.

Проект базы данных имеет защиту на уровне пользователя (Disable Shift Key).

При желании проект базы данных может модифицироваться и усовершенствоваться.
